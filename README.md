### Project Description:
This is a game of tic-tac-toe. According to Wikipedia, it is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a diagonal, horizontal, or vertical row is the winner.

I developed this project using functional programming with HTML5, CSS3, JavaScript, and jQuery.
